❤️ HeartNote

«Write from what actually happened.»

HeartNote is an emotional writing companion that helps you turn real-life moments into meaningful, structured, and deeply personal writing.

---

🌟 Overview

HeartNote is built to help you express thoughts and emotions clearly.
Instead of generic outputs, it generates context-aware, emotionally aligned writing based on what you actually experienced.

It’s not about creating content — it’s about understanding and expressing real moments.

---

🚀 Features

- ✍️ Emotion-aware writing generation
- ⚡ Real-time responses powered by AI
- 🧠 Context-based output (not generic text)
- 💾 User authentication (Signup / Login / Delete Account)
- 🗂️ Persistent database (PostgreSQL - Cloud)
- 📱 Progressive Web App (PWA support)
- 🎯 Clean and minimal UI for focused writing

---

🛠️ Tech Stack

Frontend

- HTML, CSS, JavaScript

Backend

- Python (Django)

Database

- PostgreSQL (Cloud - Render)

Other

- REST APIs
- PWA (Service Workers, Manifest)

---

🧩 Architecture

User Input → Backend (Django) → AI Processing → Response Structuring → UI Display
